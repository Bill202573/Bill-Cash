# Bill Cash — App de Controle Financeiro Pessoal

Aplicação web para controle financeiro pessoal e familiar.

## Funcionalidades

- **Dashboard** — Visão geral mensal com receitas, despesas, saldo e saúde financeira
- **Transações** — Lançamento e histórico de transações, importação de extrato
- **Contas Fixas** — Grid anual de contas fixas com controle de pagamentos e conciliação automática
- **Dívidas** — Controle de dívidas com planejador de pagamento
- **Orçamento** — Metas de gastos por categoria
- **Contas** — Gestão de contas bancárias e cartões de crédito
- **Insights** — Análises financeiras automáticas
- **Assistente IA** — Assistente financeiro pessoal com OpenAI (consulta dados, lança transações, cria objetivos)

## Stack

- **Frontend:** React 18 + TypeScript + Vite
- **UI:** TailwindCSS + shadcn/ui
- **Backend:** Supabase (PostgreSQL + Edge Functions)
- **Deploy:** Vercel

## Desenvolvimento Local

```bash
npm install
npm run dev
```

Acesse em: http://localhost:5173
